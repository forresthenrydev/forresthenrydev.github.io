MTG Mana Wizard

This is a fairly basic simulation of a mana base in a Magic: The Gathering deck. Input your deck's mana base and a Mana Value in the fields below, and the program will simulate thousands of games of your deck and output the results of how many turns it takes to be able to spend that CMC.
The simulator will construct a deck of pure lands, draw two before the game, then draw one per turn. The land to play is chosen based on which will bring us the closest to being able to spend the given mana value.

SHORTCOMINGS: As this is a simple simulation, there are a number of differences from a normal game that should be taken into account. For example, the simulator assumes that you will hit your land drop every turn. More importantly, ramp spells (such as Rampant Growth or any mana rock) are not considered.
That being said, this should be an adequeate tool for comparing one mana base to another.
